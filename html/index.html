Function reference

Top-level functions

  • eegbrowse GUI for selecting .set files for visualization
  • eegplugin_eegvis makes eegbrowse and eegvis plug in to EEGLAB menu
  • eegvis creates figure window with multi-level summary/details viewer
  • pop_eegbrowse opens an eegBrowse GUI for EEGLAB
  • pop_eegvis opens eegvis as a singleton callback for EEGLAB

Core package (+viscore)

  • blockedData manages blocked data for visualization
  • counter singleton class that returns a unique counter for IDS
  • dataConfig GUI base class for configuration
  • dataManager enhanced map for managing configuration objects
  • dataSelector container that connects configuration GUI and managed objects
  • dataSlice defines a regular subarray for data manipulation
  • managedObj base class for keyed configuration objects
  • tableConfig manages blocked data for visualization
  • tablePanel spreadsheet-like grid for editing values

Function package (+visfuncs)

  • functionConfig GUI for function configuration
  • functionObj holds definition and current values of a summary function

Property package (+visprops)

  • colorListProperty property object representing a list of colors
  • colorProperty property object representing a single color
  • configurable base class for objects with configurable properties
  • configurableObj component class for objects with configurable properties
  • doubleProperty property object representing a double in a specified interval
  • enumeratedProperty property object representing string from a list of strings
  • integerProperty property object representing an integer in a specified interval
  • intervalProperty property object representing a real interval
  • logicalProperty property object representing a logical (boolean) value
  • property base class for configurable properties
  • propertyConfig GUI for configuring properties
  • stringListProperty property object representing a cell array of strings
  • stringProperty property object representing a simple string
  • unsignedIntegerProperty property object representing an unsigned integer in a specified interval
  • vectorProperty property object representing a numeric vector

Visualization package (+visviews)

  • axesPanel base class for an axes with fixed margins
  • blockBoxPlot displays a boxplot of blocked function values by window
  • blockHistogramPlot displays histogram of block function values
  • blockImagePlot displays element vs block values as an image
  • clickable base class for mouse click linkage among view panels
  • cursorExplorable base class for objects used with cursor explorer
  • cursorExplorer class that adds and supervises an exploratory data cursor
  • dualView two level-viewer with upper panel summaries and lower panel details
  • elementBoxPlot displays a boxplot of block function values by element
  • horizontalPanel grid of horizontally arranged resizable panels
  • plotConfig GUI for configuring list of plots
  • plotObj holds definition and and settings for a plot
  • resizable interface to be implemented to give components fixed margins
  • shadowSignalPlot displays element or window signals using shadow outline
  • stackedSignalPlot displays stacked view of individual element or window signals
  • tabPanel tabbed panel for holding multiple summary views
  • verticalPanel grid of vertically arranged resizable panels
